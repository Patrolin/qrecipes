# qrecipes

## volume
```
1ml X ≈ 1g X
1ml honey ≈ 1.4g honey
 1 cup ≈ 280ml
 1 tsp ≈ 6ml
1 tbsp ≈ 18ml
```

## temperature
```
solid to liquid transition energy of water = +80°C
=> 115g -18°C ice + 300g 67°C water    // (115*(-18-80) + 300 * 67) / 400 = 22.075
   = 315g 22°C water

points
  water boils: 100°C
  milk boils:  100.1°C
  sugar melts: 186°C
```
